# CatBlaster
### 概要
文化祭のクラス企画にて展示された、3ステージ構成の弾幕stgです。  
動作が重いので、プレイする際はスペックの高いPCで遊ぶことをお勧めします。

### 操作方法
十字キーで移動（shiftキーを押しながらだと低速）、zキーでショット、xキーでボム
